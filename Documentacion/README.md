# Documentacion

La documentacion del proyecto esta organizada por dominio para facilitar busqueda y mantenimiento.

## Estructura

- `Core/` -> vision, requisitos, arquitectura, modelado, UX base y fuentes canonicas del prototipo.
- `Operacion/` -> calidad, seguridad, reglas de trabajo, colaboracion Git y setup operativo.
- `Planificacion/` -> plan de implementacion y artefactos SDD por sprint.
